Functions

jQuery.qq

Create a new queue.

Usage

var qqObj = jQuery.qq( options );

Arguments

options(Object) An object containing options specific to this queue.

Options

delay(Number) Time in milliseconds between each queue execution.  If delay is -1, queue will wait for a qqObj.next call instead of auto-executing.  Defaults to 100.
batch(Number) Number of queue items to process for each execution.  If less than this number of items exist in the queue, the remainder will be processed.  Defaults to 1.
queue(Array) Populate the queue with these items, overwriting any existing queue items.  Defaults to an empty queue.
oneach(Function) Called for each queue execution.  This callback is passed a single argument, which is the single queue item if batch is 1, or an array of queue items if batch is > 1.  If false is returned, the queue item(s) will be re-added back to the front of the queue for the next queue execution.  Inside this function, `this` represents the qq queue object.
ondone(Function) Called whenever there are no longer any queue items to process.  After completion, if more queue items are added and the queue completes again, this function will be called again.  Inside this function, `this` represents the qq queue object.
paused(Boolean) If true, start this queue in a paused state.  Defaults to false.

Returns

(Object) a reference to the qq queue object.

qqObj.add

Add a single item onto the queue.  If you want to add multiple items onto the queue individually, use qqObj.addEach.  If the queue was empty and not paused, processing will resume immediately.

Usage

qqObj.add( item [, priority ] );

Arguments

item(Anything) A single item to add to the queue.
priority(Boolean) If true, the item is added to the front of the queue, otherwise the item is added to the end of the queue.  Defaults to false.

Returns

(Number) The length of the queue, after the item has been added.

qqObj.addEach

Add multiple items onto the queue, individually.  If you want to add a single item onto the queue, use qqObj.add.  If the queue was empty and not paused, processing will resume immediately.

Usage

qqObj.addEach( items [, priority ] );

Arguments

items(Array) An array of items to add to the queue.
priority(Boolean) If true, the items are added to the front of the queue, otherwise the items are added to the end of the queue.  Defaults to false.

Returns

(Number) The length of the queue, after the items have been added.

qqObj.start

Start a currently paused queue.  If an empty queue is started, it will automatically start processing items as soon as they are added.

Usage

qqObj.start();

Returns

Nothing.

qqObj.next

Called from within a jQuery.qq oneach callback, this method will continue a queue with a delay of -1.  This is most useful for queues of asynchronous-but-serial actions, like AJAX requests that must not start until the previous has finished.

Usage

qqObj.next( [ success ] );

Arguments

success(Boolean) If false, the queue item(s) will be re-added back to the front of the queue for the next queue execution.

Returns

Nothing.

qqObj.clear

Clear a queue completely.  The paused/started status of the queue is unchanged.

Usage

qqObj.clear();

Returns

(Array) The previous queue contents.

qqObj.pause

Pause a currently running queue.  A paused but empty queue will need to be manually restarted with qqObj.start even after new items are added.

Usage

qqObj.pause();

Returns

Nothing.

qqObj.update

Update an existing queue’s options.

Usage

qqObj.update( options );

Arguments

options(Object) An object containing options specific to this queue.

Options

The delay, batch, oneach and ondone options from jQuery.qq are fully supported.

Returns

Nothing.

qqObj.size

Get the current queue length.

Usage

qqObj.size();

Returns

(Number) The length of the queue.

qqObj.indexOf

Get the current index in the queue of the passed item.

Usage

qqObj.indexOf( item );

Arguments

item(Anything) An item to test the index of.

Returns

(Number) The index of the passed item in the queue.  Returns -1 if not found.

jQuery.fn.qqAdd

Add a jQuery collection of elements onto the queue.  If you want to add each selected element onto the queue individually, use jQuery.fn.qqAddEach.  If the queue was empty and not paused, processing will resume immediately.

Usage

jQuery('selector').qqAdd( qqObj [, priority ] );

Arguments

qqObj(Object) A valid qq object, returned from jQuery.qq.
priority(Boolean) If true, the item is added to the front of the queue, otherwise the item is added to the end of the queue.  Defaults to false.

Returns

(jQuery) The initial jQuery collection of elements.

jQuery.fn.qqAddEach

Add each selected element from a jQuery collection onto the queue, individually.  If you want to add the entire jQuery collection of elements onto the queue as a single item, use jQuery.fn.qqAdd.  If the queue was empty and not paused, processing will resume immediately.

Usage

jQuery('selector').qqAddEach( qqObj [, priority ] );

Arguments

qqObj(Object) A valid qq object, returned from jQuery.qq.
priority(Boolean) If true, the items are added to the front of the queue, otherwise the items are added to the end of the queue.  Defaults to false.

Returns

(jQuery) The initial jQuery collection of elements.
